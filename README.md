# Tomato Leaf Disease Detection
Проект выполнен в рамках летней практики.

## Цель
Сравнение современных моделей обнаружения объектов для автоматического выявления заболеваний листьев томатов.

## Используемые модели
- YOLOv8
- YOLOv5
- Faster R-CNN
- SSD (планируется)
- DETR (планируется)

## Датасет
Tomato Leaf Disease Detection (Roboflow Universe)

## Используемые технологии
- Python
- PyTorch
- Torchvision
- Ultralytics YOLO
- Google Colab
- Kaggle

## Основные результаты
| Модель | Recall | mAP@0.5 | mAP@0.5:0.95 |
|--------|--------|----------|--------------|
| YOLOv8 | 0.917 | 0.971 | 0.896 |
| YOLOv5 | 0.912 | 0.968 | 0.872 |
| Faster R-CNN | 0.843 | 0.931 | 0.778 |
